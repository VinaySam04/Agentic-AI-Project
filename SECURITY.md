# Security Policy

## Supported Versions

The latest version on the default branch is supported.

## Reporting a Vulnerability

Please do not open a public issue for sensitive security concerns. Contact the maintainer
privately with a clear description, reproduction steps, and potential impact.

## API Keys

Never commit `.env` files or real API keys. Use environment variables such as
`GOOGLE_API_KEY` or `GEMINI_API_KEY`.
